# Estrategia Digital — Grupo Trimodos

Página web de presentación del plan de marketing digital para Grupo Trimodos.

## 🚀 Publicar en GitHub Pages

### Opción 1 — Subir directamente
1. Ve a [github.com/new](https://github.com/new)
2. Crea un repositorio llamado `trimodos-marketing` (público)
3. Sube `index.html` y `README.md`
4. Ve a **Settings → Pages → Source: main branch → / (root)**
5. Tu web estará en: `https://TU-USUARIO.github.io/trimodos-marketing`

### Opción 2 — Con Git (terminal)
```bash
git init
git add .
git commit -m "Lanzamiento estrategia digital Trimodos"
git remote add origin https://github.com/TU-USUARIO/trimodos-marketing.git
git push -u origin main
```
Luego activa GitHub Pages en Settings.
